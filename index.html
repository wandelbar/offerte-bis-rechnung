Über diese App

Entwickelt von Arjen Hoti — weil mich frustriert, dass für grundlegende administrative Aufgaben wie das Erstellen einer Offerte oder QR-Rechnung oft teure Software-Abos vorausgesetzt werden. Gerade für kleine Firmen, Freelancer und Kreative summiert sich das schnell zu einer unnötigen finanziellen Belastung.

Deswegen habe ich diese App entwickelt und für alle kostenlos zugänglich gemacht. Sie läuft vollständig im Browser — alle Daten werden lokal gespeichert (IndexedDB), nichts wird auf externe Server übertragen. Kein Login. Keine Installation. Keine monatlichen Kosten.

Zweck dieser Applikation

offerte-bis-rechnung.ch ist eine kostenlose, werbefreie Browser-Applikation zur Erstellung von Offertes, Auftragsbestätigungen, Lieferscheinen und Rechnungen inklusive Swiss QR-Bill. Die App richtet sich an Kleinunternehmen, Freelancer und Selbstständige in der Schweiz.

Haftungsausschluss

Die Nutzung dieser App erfolgt auf eigene Verantwortung. Für die Richtigkeit, Vollständigkeit und Aktualität der erstellten Dokumente wird keine Gewähr übernommen. Die App ersetzt keine Rechts- oder Steuerberatung.

Urheberrecht

Der Quellcode ist unter der MIT-Lizenz veröffentlicht und steht auf GitHub zur Verfügung. Inhalte, die Nutzer in der App erfassen, liegen vollständig in deren Verantwortung.

Datenschutz

Deine Daten bleiben bei dir

Diese App speichert alle Daten ausschliesslich lokal in deinem Browser (IndexedDB). Es werden keine Daten an externe Server übertragen — weder beim ersten Öffnen, noch während der Nutzung. Es gibt keinen Login, weil es keinen Server gibt, der dich authentifizieren müsste.

Was wird gespeichert?

Alle Eingaben (Firmendaten, Projekte, Dokumente, Logo) werden in der IndexedDB deines Browsers auf deinem eigenen Gerät gespeichert. Diese Daten sind für niemanden ausser dir zugänglich — nicht einmal für den Betreiber dieser App.

Was wird nicht gespeichert?

  • Keine Cookies (ausser technisch notwendiger Browser-Cache)
  • Keine Tracking- oder Analyse-Tools
  • Keine Werbung
  • Keine Nutzungsstatistiken
  • Keine Übertragung an Dritte

Externe Dienste

Die App lädt lediglich die Schriftart Inter von Google Fonts. Dabei kann Google die IP-Adresse des Nutzers sehen. Falls du das vermeiden möchtest, kannst du die App offline nutzen, nachdem sie einmalig geladen wurde (Progressive Web App).

Datenverlust / Datenlöschung

Da alle Daten ausschliesslich lokal im Browser liegen, gehen sie verloren, wenn du den Browser-Cache oder die Webseitendaten löschst. Es gibt aktuell keine automatische Backup-Funktion. Wir empfehlen, wichtige Dokumente als PDF zu exportieren.
